# ContextualSparsityLLM (2025 OPTDL Final Project)

- Group Members: Shao-Chun Ho, Yu-Chia Chang(Me)

This project investigates the phenomenon of **contextual sparsity** in Large Language Models (LLMs) to understand how different components react to dynamic parameter activation. While building upon the framework of Liu et al. (2023), the core focus is an in-depth comparative analysis of activation sparsity (density ratios). We evaluate how task complexity, model scaling, and specific architectures (Attention vs. Feed-Forward Networks) behave under aggressive sparsification across models like GPT-2, GPT-2 XL, and Llama3-8B.

For more comprehensive details, including the additive decomposition methodology, scalability analysis, and complete experimental results, please refer to the detailed project report: **[OPTDL_Final_Report.pdf](./OPTDL_Final_Report.pdf)**.
